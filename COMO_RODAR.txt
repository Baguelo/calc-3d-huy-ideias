══════════════════════════════════════════
  CALC 3D — Site estático
══════════════════════════════════════════

Este projeto agora funciona com HTML, CSS e JavaScript puro.
Não precisa Flask, Python ou servidor local para usar.

Para abrir no seu computador:

1. Abra a pasta do projeto.
2. Dê dois cliques no arquivo:

   index.html

3. O site abrirá no navegador.

══════════════════════════════════════════
  Arquivos do projeto
══════════════════════════════════════════

index.html            → interface, cálculos e botões do site
config.js             → valores fixos usados nos cálculos
static/logo-loja.jpeg → logo usada no site e no orçamento
Precificacao.py       → versão antiga em Flask, mantida como backup
CONFIGURACOES.txt     → versão antiga das configurações, mantida como backup

══════════════════════════════════════════
  O que fica salvo no config.js?
══════════════════════════════════════════

- Custo do KWh
- Valor, potência e vida útil da impressora
- Custo de consumíveis por hora
- Custos fixos mensais (aluguel, softwares, MEI, condomínio, etc.)
- Horas produtivas mensais da impressora
- Preços de tinta acrílica, primer e verniz
- Percentual estimado desses produtos consumido por peça
- Valor da mão de obra por hora de pintura
- Percentual de reserva para riscos e falhas
- Taxas da Shopee e Mercado Livre

Para editar os valores fixos, abra config.js no Bloco de Notas ou VS Code,
altere somente os números depois dos nomes e salve.

══════════════════════════════════════════
  Como publicar no GitHub Pages
══════════════════════════════════════════

1. Envie estes arquivos para o GitHub:

   index.html
   config.js
   static/logo-loja.jpeg
   COMO_RODAR.txt

2. No repositório, entre em:

   Settings → Pages

3. Em "Build and deployment", escolha:

   Source: Deploy from a branch
   Branch: main
   Folder: / (root)

4. Clique em Save.

Depois de alguns minutos, o GitHub mostrará o link público do site.
